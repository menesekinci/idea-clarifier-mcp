Intent first

Start with a fixed intent session so agents understand the goal before technical choices.

Browser Q&A

Open a local clarification UI where users select options, add custom answers, and submit once.

Structured outputs

Write decisions and planning notes as JSON that can feed scoping, architecture, and implementation.

From rough idea to usable decisions.

Idea Clarifier MCP gives agents a simple contract: clarify intent, ask decision-focused questions, review unresolved items, then write clean output files.

02 Decision questions

Small tool surface, strict flow.

start_intent_clarification

Launch the fixed first-stage intent session for new project ideas.

start_clarification

Open the second-stage product and technical decision session.

get_answers

Read submitted browser answers after the user says they are done.

write_decisions

Finalize the session into decisions.json or plan_notes.json.

Install and run.

Use the locked Python environment, then start the stdio MCP server from the repository root.

uv sync
uv run idea-clarifier
